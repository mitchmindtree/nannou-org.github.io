Quick Start - Instructions

  1. Install Rust

    2. To install Rust, open up your terminal, copy the text below, paste it into your terminal and hit enter.

    {% highlight ruby %} curl https://sh.rustup.rs -sSf | sh {% endhighlight %}
  2. Install xCode (macOS)

    3. Ensure that you have xcode-tools installed.

    Depending on what OS you are running, you might require an extra step or two.

    {% highlight ruby %} xcode-select --install {% endhighlight %}
  3. Get Nannou examples

    4. The easiest way to get familiar with Nannou is to explore the examples. To get the examples we just need to clone from the gitHub repository here

    {% highlight ruby %} git clone https://github.com/nannou-org/nannou {% endhighlight %}

    If you do not have git installed, Download it here.

    Now, change the current directory to nannou.

    {% highlight ruby %} cd nannou {% endhighlight %}
  4. Run Nannou examples

    5. Run the example using cargo.

    {% highlight ruby %} cargo run --release --example simple_window {% endhighlight %}

    The --release flag means we want to build with optimisations enabled.

    Note: If you are compiling nannou for the first time you will see cargo download and build all the necessary dependancies.
